Pagerank based Search Engine
[Aug 2018 – Nov 2018]

This project implements a search engine which retrieves the documents for a given search query and orders them based on their relevance score. The relevance score is calculated as a weighted sum of different query related criteria, one of which is the page rank of a document (calculated as per original page rank algorithm). Based on this relevance score, top 10 most relevant documents are returned as the results to the given search query.
